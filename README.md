# ML automation with MLflow

クラウド上へのデータアップロードをトリガーとして、モデル学習とその学習結果の管理・可視化を一気通貫で行うシステムを作成しました。  
AWS上でストレージやサーバを管理しその中でMLflowを実装しています。

```
+------------------+        +--------------+        +--------------+
|                  |        |              |        |              |
|  Amazon S3       +------->+ AWS Lambda   +------->+ Amazon EC2   |
|  (Dataset        | Event  |  (Trigger)   | Start  | (Training &  |
|  Storage)        |        |              | EC2    |  MLflow)     |
|                  |        |              |        |              |
+------------------+        +-------+------+        +------+-------+
                                   |                      |
                                   |                      |
                                   v                      |
                            +--------------+              |
                            |              |              |
                            | Amazon RDS   |<-------------+
                            |  (MLflow     | Log Results
                            |  Backend)    |
                            |              |
                            +--------------+

```
